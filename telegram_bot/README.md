# Telegram Network Monitor Bot

Бот для мониторинга роутеров и сканирования сети/устройств с управлением через Telegram.

## Возможности

- Мониторинг статуса роутеров по портам 8080/8022 с уведомлениями о смене статуса.
- Сканирование сети на любые устройства с определением типа (роутер, свитч, майнер, камера, телефон и др.).
- Сканирование сети только на майнеры (Antminer, Avalon, Whatsminer и др.) с выводом hashrate и uptime.
- Загрузка CSV-файла со списком IP для сканирования как майнеров.
- Вывод результатов в чат и возможность получить файл с результатами (CSV).

## Установка

1. Клонируйте репозиторий или скопируйте файлы в нужную папку.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Вставьте токен Telegram-бота в файл `bot/config.py`:
   ```python
   TELEGRAM_BOT_TOKEN = 'ВАШ_ТОКЕН_ТУТ'
   ```

## Запуск

```bash
python -m bot.main
```

## Использование

### Главное меню
- **Статус роутеров** — проверяет доступность роутеров 11.250.0.1–11.250.0.5 по портам 8080/8022.
- **Сканировать сеть** — запросит сеть/маску (например, 192.168.1.0/24), просканирует все устройства, определит типы, выведет в чат и по запросу отправит CSV-файл.
- **Загрузить файл для сканирования** — отправьте CSV-файл с колонкой `ip`, бот просканирует их как майнеры и выведет результат.
- **Сканировать майнеров** — запросит сеть/маску, просканирует только майнеры (порт 4028), выведет статус, hashrate, uptime и по запросу отправит CSV-файл.

### Формат CSV для загрузки

```
ip
192.168.1.10
192.168.1.11
...
```

## Размещение на сервере
- Бот работает на Python 3.8+.
- Для работы в фоне используйте supervisor, systemd или tmux/screen.

## Важно
- Для сканирования сети и портов требуются соответствующие права (запуск от имени пользователя с доступом к сети).
- Для больших сетей сканирование может занять значительное время.
