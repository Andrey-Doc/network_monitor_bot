# Telegram Bot для мониторинга и сканирования

Telegram бот для мониторинга роутеров, сканирования сетей и поиска майнеров с расширенными возможностями настройки.

## 🚀 Возможности

### Основные функции
- **🌐 Мониторинг роутеров** - проверка статуса роутеров по IP и портам
- **🔍 Сканирование сетей** - поиск устройств в указанной сети
- **⛏️ Поиск майнеров** - специализированное сканирование для майнеров (порт 4028)
- **⚡ Быстрое сканирование** - параллельное сканирование для ускорения
- **📊 Статистика** - отслеживание использования и результатов
- **🔔 Уведомления** - система уведомлений с уровнями важности
- **⚙️ Настройки** - полноценная система управления настройками

### Система настроек
Бот включает мощную систему настроек с возможностью изменения всех параметров:

#### 🌐 Мониторинг

- Автозапуск мониторинга
- Уведомления об изменениях статуса

- Включение/выключение уведомлений
- Тихие часы (настраиваемое время)

- Порты для сканирования
- Специальные порты для майнеров и роутеров
- TTL результатов сканирования

#### 🌐 Роутеры
- IP адреса роутеров для мониторинга
- Порты роутеров для проверки
- Интервал проверки роутеров
- Статус роутеров

#### 🎨 Интерфейс
- Язык интерфейса
- Отображение прогресса сканирования
- Отображение временных меток
- Компактный режим

#### 🔒 Безопасность
- Список разрешенных пользователей
- Ограничение настроек только для администраторов
- Уровень логирования
- Управление токеном бота

- Интервал создания резервных копий

## 📋 Команды

### Основные команды
- `/start` или `/menu` - главное меню
- `/status` - статус активных сканирований
- `/stats` - статистика использования
- `/help` - справка по командам

### Команды мониторинга
- `/monitor_start` - запуск фонового мониторинга
- `/monitor_stop` - остановка мониторинга

### Команды экспорта
- `/export_stats` - экспорт статистики в CSV

## ⚙️ Настройка

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Конфигурация
1. Создайте файл `telegram_bot/data/settings.json`:
```json
{
  "routers": {
    "ips": ["11.250.0.1", "11.250.0.2", "11.250.0.3", "11.250.0.4", "11.250.0.5"],
    "ports": [8080, 8022, 80, 22]
  },
  "scanning": {
    "default_timeout": 5,
    "max_concurrent_scans": 3,
    "results_ttl": 3600
  },
  "monitoring": {
    "auto_start": true,
    "interval": 300
  }
}
```
2. Создайте файл `telegram_bot/data/secrets.json`:
```json
{
  "TELEGRAM_BOT_TOKEN": "your-bot-token-here",
  "CHAT_ID": 123456789
}
```

### Запуск
```bash
python -m telegram_bot.bot.main
```

## 🎯 Система настроек

### Доступ к настройкам
Настройки доступны через кнопку "⚙️ Настройки" в главном меню бота.

### Категории настроек

#### 🌐 Мониторинг
- **Интервал мониторинга**: от 30 секунд до 1 часа
- **Автозапуск**: автоматический запуск мониторинга при старте бота
- **Уведомления об изменениях**: отправка уведомлений при изменении статуса роутеров
- **Уведомления при запуске**: уведомления о запуске системы мониторинга

#### 🔔 Уведомления
- **Включение/выключение**: общее управление уведомлениями
- **Тихие часы**: период времени без уведомлений
- **Уровни уведомлений**: настройка важности уведомлений
- **Время тихих часов**: настройка периода тишины

#### 🔍 Сканирование
- **Таймаут сканирования**: время ожидания ответа от устройства
- **Макс. параллельных сканирований**: количество одновременных сканирований
- **Порты для сканирования**: список портов для проверки
- **Порты майнеров**: специальные порты для поиска майнеров
- **Порты роутеров**: порты для проверки роутеров
- **TTL результатов**: время хранения результатов сканирования

#### 🌐 Роутеры
- **IP адреса роутеров**: список IP для мониторинга
- **Порты роутеров**: порты для проверки роутеров
- **Интервал проверки**: частота проверки роутеров
- **Статус роутеров**: текущее состояние роутеров

#### 🎨 Интерфейс
- **Язык интерфейса**: выбор языка бота
- **Показывать прогресс**: отображение прогресса сканирования
- **Показывать время**: отображение временных меток
- **Компактный режим**: сокращённый вывод информации

#### 🔒 Безопасность
- **Разрешенные пользователи**: список пользователей с доступом
- **Только админ настройки**: ограничение настроек для администраторов
- **Уровень логирования**: детализация логов
- **Изменить токен**: обновление токена бота

#### 💾 Резервное копирование
- **Авторезервное копирование**: автоматическое создание резервных копий
- **Интервал резервного копирования**: частота создания копий
- **Макс. количество резервных копий**: лимит хранимых копий
- **Создать резервную копию сейчас**: ручное создание копии

#### 📊 Экспорт/Импорт
- **Экспорт настроек**: сохранение настроек в JSON
- **Импорт настроек**: загрузка настроек из JSON
- **Экспорт статистики**: сохранение статистики
- **Экспорт логов**: сохранение логов

### Управление настройками

#### Просмотр настроек
- **Сводка**: краткий обзор всех настроек
- **Детальный просмотр**: по категориям

#### Изменение настроек
- **Через кнопки**: для простых настроек (вкл/выкл, выбор из списка)
- **Через ввод**: для числовых значений и списков

#### Сброс настроек
- **Сброс к значениям по умолчанию**: восстановление исходных настроек

#### Экспорт/Импорт
- **Экспорт**: сохранение текущих настроек
- **Импорт**: загрузка настроек из файла

## 🔧 Технические детали

### Асинхронное сканирование
- Использование `asyncio` для неблокирующего сканирования
- Поддержка параллельного и последовательного сканирования
- Прогресс-бары в реальном времени

### Система уведомлений
- Уровни важности: info, warning, critical, success
- Типы уведомлений: system_alert, scan_completed, scan_error, router_status_change
- Поддержка тихих часов

### Статистика
- Отслеживание команд пользователей
- Статистика сканирований
- Статистика роутеров
- Экспорт в CSV

### Система настроек
- JSON-файл для хранения настроек
- Автоматическое создание настроек по умолчанию
- Валидация значений
- Экспорт/импорт настроек
- Сброс к значениям по умолчанию
