# Telegram Bot для мониторинга и сканирования

Бот для мониторинга роутеров, сканирования сетей и поиска майнеров с расширенными возможностями.

## 🚀 Новые возможности

### 1. Фоновый мониторинг роутеров
- **Автоматическое отслеживание** статуса роутеров в фоновом режиме
- **Уведомления об изменениях** статуса в реальном времени
- **Настраиваемый интервал** проверки (по умолчанию 5 минут)
- **Отслеживание изменений** доступности и открытых портов

### 2. Система уведомлений
- **Разные уровни важности**: INFO, WARNING, CRITICAL, SUCCESS
- **Типизированные уведомления**: статус роутеров, сканирование, ошибки, отчёты
- **Очередь уведомлений** для надёжной доставки
- **Автоматические уведомления** о запуске/остановке бота

### 3. Inline-кнопки
- **Современный интерфейс** с inline-кнопками вместо обычных
- **Быстрый доступ** к основным функциям
- **Интерактивные меню** для навигации
- **Удобное управление** мониторингом

### 4. Система статистики
- **Подробная статистика** использования бота
- **Отслеживание активности** по дням
- **Экспорт данных** в CSV формат
- **Автоматические отчёты** о производительности

## 📋 Основные функции

### Мониторинг роутеров
- Проверка статуса роутеров (IPs: 11.250.0.1-11.250.0.5)
- Мониторинг портов 8080 и 8022
- Фоновый мониторинг с уведомлениями

### Сканирование сетей
- **Полное сканирование**: поиск всех устройств
- **Быстрое сканирование**: параллельное сканирование
- **Сканирование майнеров**: поиск устройств на порту 4028
- **Загрузка файлов**: сканирование IP из CSV файлов

### Работа с майнерами
- Автоматическое определение майнеров
- Получение hashrate и uptime через API
- Экспорт результатов в CSV

## 🛠 Установка и настройка

### Требования
```bash
pip install -r requirements.txt
```

### Настройка
1. Создайте файл `telegram_bot/bot/config.py`:
```python
TELEGRAM_BOT_TOKEN = "ваш_токен_бота"
CHAT_ID = "ваш_chat_id"
ROUTER_IPS = ["11.250.0.1", "11.250.0.2", "11.250.0.3", "11.250.0.4", "11.250.0.5"]
ROUTER_PORTS = [8080, 8022]
SCAN_RESULTS_TTL = 3600  # время жизни результатов в секундах
```

### Запуск
```bash
python telegram_bot/run_bot.py
```

## 🔧 Настройка мониторинга

### Запуск фонового мониторинга
```python
# Автоматически при запуске бота или через команду
/monitor_start
```

### Настройка интервала
```python
# В background_monitor.py
await background_monitor.start_monitoring(interval=300)  # 5 минут
```

### Получение уведомлений
Бот автоматически отправляет уведомления в указанный CHAT_ID при:

- Завершении сканирований
- Возникновении ошибок
- Запуске/остановке системы
